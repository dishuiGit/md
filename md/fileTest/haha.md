### 定义函数
```
//x-->输入参数,Double-->输入类型
scala> def square(x:Double):Double = x*x
square: (x: Double)Double

scala> square(3)
res1: Double = 9.0
```

```
//同上
scala> def square(x:Double) = x*x

```

### if else
```
scala> def abs(x:Double) = if (x>=0) x else -x
abs: (x: Double)Double

scala> abs(34)
res2: Double = 34.0

scala> abs(-34)
res3: Double = 34.0
```
